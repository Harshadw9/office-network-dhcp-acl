# Small Office Network with DHCP & ACL Security

## Overview
This project demonstrates a secure small office network using Cisco Packet Tracer. The network uses VLANs for department separation, DHCP for automatic IP assignment, and ACLs for access control security.

## Technologies Used
- Cisco Packet Tracer
- VLAN
- DHCP
- ACL
- Routing & Switching
- TCP/IP

## VLAN Information

| Department | VLAN | Network |
|------------|------|----------------|
| HR         | 10   | 192.168.10.0/24 |
| Sales      | 20   | 192.168.20.0/24 |
| Admin      | 30   | 192.168.30.0/24 |

## Devices Used
- Cisco 2811 Router
- Cisco 2950 Switch
- 6 PCs

## Features
- VLAN Configuration
- Inter-VLAN Routing
- DHCP Configuration
- ACL Security
- Automatic IP Assignment
- Connectivity Testing

## Security Rule
Sales department is restricted from accessing the Admin department using ACL configuration.

## Outcome
Successfully implemented a secure office network with DHCP-based IP allocation and ACL-based access control.
